# Código de Conduta

Este projeto adota o Contributor Covenant. Esperamos que todos os participantes mantenham um ambiente respeitoso e colaborativo.

## Nosso compromisso

Estamos comprometidos em proporcionar uma comunidade acolhedora, livre de assédio e discriminação.

## Padrões de comportamento

- Seja respeitoso e construtivo.
- Considere diferentes pontos de vista.
- Aceite críticas de forma educada.

## Aplicação

Comportamentos inadequados podem ser reportados via issues ou contato direto com o mantenedor.

Referência: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
