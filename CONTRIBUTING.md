# Contribuindo

Obrigado por considerar contribuir com este projeto!

## Como contribuir

1. Faça um fork do repositório.
2. Crie uma branch a partir de `main`.
3. Faça as alterações com foco em melhorias claras e objetivas.
4. Execute os checks locais:
   - `npm install`
   - `npm run lint`
   - `npm test`
5. Abra um Pull Request descrevendo o que mudou e por quê.

## Padrões

- Use commits pequenos e claros.
- Mantenha a estrutura em `src/` e testes em `tests/`.
- Formate o código com `npm run format` quando necessário.
